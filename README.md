# geoSphere

## Getting Started
1. Install MATLAB
2. Download & Install [GIBBON](https://github.com/gibbonCode/GIBBON) library
    - [How to install](https://github.com/gibbonCode/GIBBON#installation-)
3. Run **makeGeoSphere.m**
